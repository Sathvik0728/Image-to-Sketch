# 🖼️ Pencil Sketch Generator

A modern web application that converts normal images into realistic pencil sketch drawings using Python, Flask, and OpenCV.

---

# ✨ Features

✅ Realistic pencil sketch effect  
✅ Beautiful modern dark UI  
✅ Instant image preview  
✅ Download sketch image  
✅ Responsive design  
✅ Fast image processing  
✅ No permanent image storage  
✅ Simple and clean project structure  

---

# 🚀 Technologies Used

- Python
- Flask
- OpenCV
- NumPy
- HTML
- CSS

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Pencil-Sketch-Generator.git
```

---

## 2️⃣ Open Project Folder

```bash
cd Pencil-Sketch-Generator
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

---

## 5️⃣ Install Required Packages

```bash
pip install -r requirements.txt
```

---

---

# 💡 How It Works

1. Upload an image  
2. Image is converted to grayscale  
3. OpenCV applies blur and inversion  
4. Pencil sketch effect is generated  
5. Result displayed instantly
